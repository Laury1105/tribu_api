# tribu api

## Project setup

```
Cloner le projet et le placer dans un serveur local (Mamp, wamp, xampp,...) dans le dossier htdocs ou www (à voir en fonction du serveur) et modifier le chemin du 'document root' vers le projet
ex : "Applications ▹ MAMP ▹ htdocs ▹ tribu_api"
```
```
Paramètres du serveur local : 
Apache Port 8081
MySql Port 8889
```
```
Importer la DB dans mysql à l'aide du fichier DB.txt (nom de la DB : 'api')
```
```
Lancer le serveur
```
